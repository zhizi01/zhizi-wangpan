# ZHIZI 网盘

一个基于 Express 开发的云端文件管理系统，结合 Linear Design 设计风格，提供个人文件管理、文件共享、在线分享等核心功能，适配轻量云服务器部署。

## 📖 项目说明

本项目全程由 kimi 家族模型开发实现：

- 项目基础框架由 kimi2\.6 模型构建

- 因 kimi2\.6 额度限制，后续 BUG 修复采用基于 kimi2\.5 训练的 composer2 模型完成

## 🎯 核心需求与设计

### 基础环境适配

- 服务器配置：2H4G 轻量云服务器

- 本地开发环境：Node\.js 24\.x

- 数据库：MySQL（配置信息统一放在 \.env 文件中，便于部署修改）

- 前端设计：基于 VITE 构建，采用 Linear Design 设计语言，界面简洁、交互流畅

### 核心功能

1. **认证系统**

    - 支持注册开关配置（可设置允许/禁止新用户注册）

    - 账号以邮箱作为唯一标识

    - 邮箱验证码登录/注册，基于 SMTP 服务发送验证码

2. **个人文件空间**：每个用户拥有独立的文件存储空间，支持文件上传、下载、删除、移动、重命名等基础操作

3. **文件共享广场**：用户可将文件分享至公共广场，供其他用户查看、下载（权限可控）

4. **文件在线分享**：生成专属分享链接，支持两种分享模式——需登录才可下载 / 无需登录直接下载，灵活适配不同场景
